3 总结

  • 多上 GitHub 看看原仓库的文档、issues、PR，会有收获的，实在不行直接搜，比百度好使。
  • 熟悉流程，Hexo 的原理是根据渲染器，把 Markdown 语法转为 HTML 语法。 所以一些显示 bug 不一定是主题的问题，而很有可能是渲染器的问题。所以看看渲染器的源码总是有收获的。
